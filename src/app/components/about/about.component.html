About Angular Text Extractor

Features

  • Image OCR: Extract text from images using Tesseract.js OCR engine
  • PDF Text Extraction: Extract text directly from PDF documents
  • Multiple File Formats: Supports various image formats (PNG, JPG, JPEG, etc.) and PDF files
  • Confidence Scoring: Get OCR confidence scores for extracted text
  • History Tracking: Keep track of all your extractions
  • Export Options: Copy to clipboard or download as text file

Technologies Used

Angular 17 - Modern web framework
Tesseract.js - OCR engine for image text extraction
PDF.js - PDF text extraction library
TypeScript - Type-safe JavaScript

How to Use

  1. Navigate to the Text Extractor page
  2. Click the file input and select an image or PDF file
  3. Wait for the text extraction to complete
  4. View, copy, or download the extracted text

Component Architecture

This application demonstrates a generic, reusable component pattern:

  • TextExtractorComponent: Generic component that handles file upload and emits extracted text
  • TextExtractorParentComponent: Parent component that uses the generic component and adds additional features
  • TextExtractorService: Service that handles the actual text extraction logic
